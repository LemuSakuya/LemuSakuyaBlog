# 08 综合实验与调试方法

## 1. 综合实验推荐题

1. 按键设置参数，数码管显示参数值
2. 定时器周期采样温度并串口上传
3. 超阈值时蜂鸣器报警 + LED 提示

## 2. 分层设计模板

- bsp 层：GPIO、Timer、UART 驱动
- device 层：LED、Key、Sensor 封装
- app 层：业务状态机与任务调度

## 3. 常见调试方法

1. 串口打印关键变量
2. 先测单模块，再做联调
3. 给每个状态加可观测输出（LED 或日志）
4. 出现异常先查时序，再查硬件连接

## 4. 报告写作建议

- 问题分析：需求拆解 + 状态图
- 硬件设计：端口分配 + 原理图
- 软件设计：流程图 + 关键函数
- 测试结果：截图 + 现象说明 + 误差分析
- 总结：遇到的问题与改进方向

## 5. 从 51 迁移到 STM32 的最小路径

1. 保留状态机思维
2. 用 HAL 替代寄存器裸操作
3. 将 delay 方案迁移为 SysTick/定时器节拍
4. 逐步引入 FreeRTOS 任务调度
